# Finance App MVP

Flutter MVP skeleton for a personal finance app with:

- account tracking
- monthly budgets
- transaction logging
- investment and retirement summaries
- simple AI-ready report payloads

## Current state

This project was scaffolded manually because Flutter is not installed in the current environment.

## Recommended next steps

1. Install Flutter SDK.
2. From this folder, run `flutter create .`
3. Review generated platform folders and keep the existing `lib/` code.
4. Run `flutter pub get`
5. Start building local persistence with `SQLite + Drift`

## Implemented in this skeleton

- domain models for accounts, categories, budgets, transactions, and asset snapshots
- in-memory sample repository
- dashboard, accounts, budgets, transactions, reports tabs
- account grouping logic aligned with your finance design
- drift database schema and bootstrap path for SQLite persistence
